## ⚠️ Известные проблемы и решения

### Проблема: "Не удается зарегистрироваться"

**Причины:**
1. Бэкенд не запущен → **Решение**: Запустите `npm run dev` в папке `server/`
2. Письмо находится в спаме → **Решение**: Проверьте спам-папку в почте
3. Переменные окружения не загружены → **Решение**: Перезагрузите браузер и терминалы
4. Интернет-соединение → **Решение**: Проверьте connection к Supabase

### Проблема: "ECONNRESET при npm install"

**Причина:** Сетевая проблема  
**Решение:**
```bash
npm install --legacy-peer-deps
# или
npm cache clean --force
npm install
```

### Проблема: Axios версия не обновляется

**Причина:** `npm install` не выполнен после обновления `package.json`  
**Решение:**
```bash
cd server
rm -rf node_modules package-lock.json
npm install
```
